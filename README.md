## Route List

#### Just list of my route

```http
  php artisan route:list
```

| Method     | URI            | Name                       |
| :--------  | :------------- | :------------------------- |
| `GET/HEAD` | `/`            | index                      |
| `GET/HEAD` | `create`       | create                     |
| `GET/HEAD` | `destroy/{id}` | destroy                    |
| `GET/HEAD` | `edit/{id}`    | edit                       |
| `POST`     | `store`        | store                      |
| `PUT`      | `update/{id}`  | update                     |
